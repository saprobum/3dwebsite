This page helps you test if the mobile menu is working correctly.

Quick Tests:

Instructions:

  1. Open your actual website (index.html, about.html, etc.)
  2. Open browser DevTools (F12)
  3. Go to Console tab
  4. Resize window to mobile size (< 576px width)
  5. Click the hamburger menu
  6. Check console for "Menu toggled" message
  7. Inspect the #navigation element to see if .show class is added

Common Issues:

  • Menu not visible: Check if CSS file is loaded
  • Click not working: Check browser console for JavaScript errors
  • Class not toggling: Verify element ID is "navigation"
  • CSS not applying: Clear browser cache (Ctrl+Shift+R)
